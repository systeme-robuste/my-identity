Bienvenue dans la documentation de **My Identity**, la plateforme no-code construite pour la prochaine décennie.

## Démarrage rapide

1. Créez un compte sur [myidentity.app](https://myidentity.app)
2. Créez votre premier site
3. Ajoutez des blocs depuis la palette
4. Connectez un domaine personnalisé
5. Publiez

## Architecture

My Identity est construit sur :

- **Cloudflare Workers** — rendu edge mondial
- **Neon Postgres** — base de données serverless
- **Drizzle ORM** — typage end-to-end
- **Stripe** — paiements et abonnements
- **Resend** — emails transactionnels
- **Mistral AI** — workflows IA

## Pourquoi My Identity ?

- **0 JavaScript** côté client par défaut (sites ultra-rapides)
- **USD uniquement** — prix stables
- **RGPD-compliant** par conception
- **Open source** (MIT)
